## Environment variables

Set these variables in your shell, CI pipeline, or container runtime. Spring Boot automatically maps `SPRING_DATASOURCE_*` environment variables to `spring.datasource.*` settings:

- `SPRING_DATASOURCE_URL` - JDBC URL for the MySQL database.
- `SPRING_DATASOURCE_DRIVER_CLASS_NAME` - JDBC driver class name.
- `SPRING_DATASOURCE_USERNAME` - Database username.
- `SPRING_DATASOURCE_PASSWORD` - Database password.
